ABOUT THE GAME

  • Patuti is an alien character who is being accused of being a hostile creature by the public.

  • Try to save Patuti by tapping/clicking the screen so he can jump and avoid the approaching bullets. You can earn points for each successful bullet evasion by Patuti. Share your results with others and become the King of Patuti Jump!

  • This is a simple HTML, CSS, and JavaScript project for the course Project made with Bootstrap for responsive Web Layout for both Mobile and Desktop devices. IT461 Advanced Web Systems.
